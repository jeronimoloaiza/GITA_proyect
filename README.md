# Graphene Layer Segmentation and Contrast Analysis

Este repositorio integra tanto el **trabajo experimental** de obtención de grafeno mediante exfoliación micromecánica como el **pipeline computacional** para la segmentación y análisis de imágenes ópticas de flakes en sustratos de silicio con óxido (Si/SiO₂).

---

## 📖 Contenido
- 📄 **Paper experimental**:  
  *Obtención y Caracterización Preliminar de Grafeno en Substratos de Silicio Mediante Exfoliación Micromecánica*  
  Juan Pablo González Blandón, Juan José Balvin Torres (2025).

- 💻 **Código en Python**:
  - Segmentación automática de regiones de grafeno en imágenes ópticas.
  - Cálculo de intensidades y contraste local con respecto al sustrato.
  - Estimación preliminar del número de capas (1, 2, 3+) en función del contraste óptico.
