# Graphene Layer Segmentation and Contrast Analysis

This repository integrates both the **experimental work** on obtaining graphene via micromechanical exfoliation and the **computational pipeline** for segmentation and optical image analysis of graphene flakes on silicon substrates with oxide (Si/SiO₂).

---

## 📖 Contents
- 📄 **Experimental Paper**:  
  *Preliminary Production and Characterization of Graphene on Silicon Substrates via Micromechanical Exfoliation*  
  Juan Pablo González Blandón, Juan José Balvin Torres (2025).

- 💻 **Python Code**:
  - Automatic segmentation of graphene regions in optical microscopy images.
  - Calculation of intensities and local contrast with respect to the substrate.
  - Preliminary estimation of the number of layers (1, 2, 3+) based on optical contrast.
